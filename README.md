## GETTING STARTED 



### Requirements:

 * __Node__: ^14.17.6
 * __NPM__: ^6.14.16 
 

### Installation:

1. Clone the repo


```sh
git clone https://github.com/LavalAlex/battleship.git
```


2. Install NPM packages

* client folder

```sh
npm install
```
